CSS Box model summary

The ways Box model Works

Box model have a content and sizing. IN the constent and sizing boxes have different behavior based on their display value, their set dimension, and the content that live within them. The boxes can be control by using extrinsic sizing , or, you can continue to let the browser make decisions for you based on the content size, using intrinsic sizing, there are two type of sizing, Extrinsic sizing and intrinsic sizing.

Box model areas

Boxes are made up off distinct box model areas that do their job.The box model areas start with a box called content box,which is the area the the constent live in.There is Another box name padding box which surronds the content box and is the space created by the padding property.The border box surrounds the padding box and its space is occupied by the border value. The border box is the bounds of your box and the border edge is the limit of what you can visually see. The border property is used to visually frame an element.The margin box,is the space around your box, defined by the margin rule on your box.

How the box model is controlled

The box model is controlled in the browser.The CSS used varies between each browser, but they provide sensible defaults to make content easier to read.Inline-block are used and thoso elements will resptect the block margin, while the element maintains most of the same behaviors it had as an inline element.

Debugging the box model

Visualisation of a selected box should be provided. It can help you understand how box model works and.There is a browser Devtools that provide the visualisation.
